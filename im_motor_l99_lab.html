Power-Flow Identities (Algebraic Model)

Input power:
P_in = √3 · V_LL · I_L · cos φ

Air-gap power:
P_elm = P_in − ΔP₁w − ΔP_Fe

Rotor copper loss identity (KEY):
ΔP₂w = s · P_elm  ⇒  s = ΔP₂w / P_elm

Mechanical (developed) power:
P_m = P_elm − ΔP₂w = (1−s) · P_elm

Shaft (output) power:
P_out = P_m − ΔP_rot − ΔP_str

Synchronous speed:
n_s = 120·f / P  ,  ω_s = 2π·n_s/60

Air-gap (electromagnetic) torque:
T_elm = P_elm / ω_s  =  P_m / ω_m

Shaft torque:
T_sh = P_out / ω_m

Dynamic Model (d-q reference frame)

Stator voltage equations:
v_ds = R_s·i_ds + dλ_ds/dt − ω_e·λ_qs
v_qs = R_s·i_qs + dλ_qs/dt + ω_e·λ_ds

Rotor voltage equations (squirrel cage, v_dr=v_qr=0):
0 = R_r·i_dr + dλ_dr/dt − (ω_e−ω_r)·λ_qr
0 = R_r·i_qr + dλ_qr/dt + (ω_e−ω_r)·λ_dr

Flux linkages:
λ_ds = L_s·i_ds + L_m·i_dr
λ_qs = L_s·i_qs + L_m·i_qr
λ_dr = L_r·i_dr + L_m·i_ds
λ_qr = L_r·i_qr + L_m·i_qs

Electromagnetic torque:
T_e = (3/2)·(P/2)·(λ_ds·i_qs − λ_qs·i_ds)

Mechanical equation:
J·dω_m/dt = T_e − T_L − B·ω_m
ω_r = (P/2)·ω_m (rotor electrical speed)

Steady-State Equivalent Circuit (Per-Phase)

Thévenin form used for torque-slip:V_th = V_s · X_m / √(R_s²+(X_s+X_m)²)
Air-gap power:P_elm = 3·I_r²·R_r/s  |  Slip torque:T = P_elm/ω_s

Phasor Diagram (Stator)

Phase voltage V_ph reference (real axis). Stator current I lags by φ = arccos(0.85). Vector decomposition: I·cosφ (active) and I·sinφ (reactive).
